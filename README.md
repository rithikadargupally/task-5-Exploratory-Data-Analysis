# Titanic Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset.

The goal is to understand the dataset, clean the data, analyze passenger information, and find patterns related to survival.

## Dataset
The dataset contains passenger details from the Titanic ship, including:

- Passenger Class
- Gender
- Age
- Fare
- Survival Status
- Family information

## Tools & Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed

1. Loaded the Titanic dataset
2. Checked dataset structure and information
3. Analyzed missing values
4. Checked duplicate records
5. Performed statistical analysis
6. Created visualizations:
   - Histogram
   - Boxplot
   - Scatter plot
   - Correlation heatmap
   - Pairplot
7. Studied survival patterns

## Key Findings

- Female passengers had higher survival rates.
- Passenger class affected survival chances.
- Higher-class passengers had better survival rates.
- Some columns contained missing values.
- Age and fare showed variations among passengers.

## Files Included

- `Titanic_EDA.ipynb` - Jupyter Notebook containing code and analysis
- `Titanic Exploratory Data Analysis.pdf` - Project report

## Conclusion

The analysis helped understand the factors affecting passenger survival and demonstrated the process of performing EDA using Python.
